====== LU01b - Verweise und Anker ======

===== Lernziele =====
  - Ich kann Verweise (Hyperlinks) in einer Webseite hinzufügen und anhand der Attribute (z.B. href, id) anpassen kann. HTML-Elemente/Tags sind <a>.
  - Ich kenne bei den Verweisen den Unterschied zwischen absoluter und relativer Verweis.
  - Ich kann Verweise syntaktisch korrekt erstellen und kann Merkmale und Eigenschaften für den korrekten Aufbau aufzählen.
  - Ich kenn den Zweck von lokalen Verweisen (Anker).
  - Ich kann lokale Verweise (Anker) korrekt erstellen.

===== Einleitung =====

Einen Link können Sie in HTML mit dem <color #ed1c24><a></color>-Element (a = anchor, deutsch: Anker) erstellen. Der Linktext (oder Verweistext) zwischen den <a>-Tags wird aktiviert, wenn das href-Attribut einen validen Wert hat.

==== Beispiel ====
Ein Linktext wird gewöhnlich vom Webbrowser (meistens in Blau) unterstrichen. Dies können Sie aber (wie immer) mit CSS jederzeit ändern.

<code><a href="http://www.bzz.ch">Bildungszentrum Zürichsee</a></code>

===== Links zu anderen HTML-Dokumenten der eigenen Webseite einfügen =====
Die Verweise verbinden die losen Sammlungen von HTML-Dokumenten zu einer zusammenhängenden Webseite. Daraus lässt sich die sog. Navigation der Webseite erstellen. Bei einem Link zu einer anderen Seite derselben Webseite genügt es, die relative (statt der absoluten) URL anzugeben. Dazu später mehr.

==== Beispiel ====

Folgende in der Abbildung abgedruckte Verzeichnisstruktur sei als Beispiel gegeben.

{{de:modul:m287:learningunits:lu01:02.png?600|}}

Die Verlinkung für die Startseite, hier index.html, zu den anderen Seiten <color #ed1c24>links.html</color>, <color #ed1c24>about.html</color> und <color #ed1c24>impressum.html</color> sieht demnach in der Praxis wie folgt aus:

<code html>
<!doctype html>
<html lang="de">
  <head>
    <title>Startseite - Blog</title>
    <meta charset="UTF-8">
  </head>
  <body>
    <nav>
      Blog |
      <a href="seiten/links.html">Links</a> |
      <a href="seiten/about.html">Über mich</a> |
      <a href="seiten/impressum.html">Impressum</a>
    </nav>
    <h1>Mein Blog</h1>
    <p>Neueste Berichte zu HTML</p>
  </body>
</html>
</code>

---

Natürlich müssen Sie auch die Links zu den anderen Seiten, also <color #ed1c24>links.html</color>, <color #ed1c24>about.html</color> und <color #ed1c24>impressum.html</color> anpassen. Dabei sollen die relativen URL angegeben werden. Bezogen auf die Seite <color #ed1c24>links.html</color>, würden die Angaben für das Attribut href wie folgt aussehen:

<code html>
<!doctype html>
<html lang="de">
  <head>
    <title>Linksammlung</title>
    <meta charset="UTF-8">
  </head>
  <body>
    <nav>
      <a href="../index.html">Blog</a> |
      Links |
      <a href="about.html">Über mich</a> |
      <a href="impressum.html">Impressum</a>
    </nav> 
    <h1>Linksammlung</h1>
    <p>Interessante Linksammlung zu HTML-Themen</p>
  </body>
</html>
</code>

Über den Unterordner seiten navigieren Sie mit <color #ed1c24>../</color> (hier <color #ed1c24>../index.html</color>) zum übergeordneten Ordner, wo sich <color #ed1c24>index.html</color> befindet. Die anderen beiden Dateien <color #ed1c24>about.html</color>, <color #ed1c24>impressum.html</color> befinden sich im selben Ordner wie <color #ed1c24>links.html</color>, daher reicht es aus, nur den Dateinamen anzugeben. Ähnlich müssen Sie auch die Dateien <color #ed1c24>about.html</color> und <color #ed1c24>impressum.html</color> verlinken.

===== Links zu anderen Webseiten einfügen =====

Links zu anderen Webseiten werden genauso notiert wie die Links zu den Seiten derselben Webseite. Der Unterschied liegt in der absolute URL (also die Angabe der kompletten Adresse) im Attribut href.

<code>
<nav>
  <ul>
    <li><a href="http://www.w3.org/html/wg/drafts/html/master/">HTML 5.1 Nightly</a></li>
    <li><a href="http://www.w3.org/">W3C</a></li>
    <li><a href="http://www.whatwg.org/">WHATWG</a></li>
  </ul>
</nav>
</code>

===== Links in einem neuen Fenster öffnen ===== 

Mit dem HTML-Attribut <color #ed1c24>target="_blank"</color> wird der Verweis in einem neuen Fenster oder Tab geöffnet wird.
Ziel der Verwendung von <color #ed1c24>target="_blank"</color> ist den Besucher der Seite nicht zu verlieren, sondern die Seite offen zu lassen, damit er dort zurückkehrt, wenn er die Seite im neu geöffneten Fenster oder Tab gelesen hat.

==== Beispiel ====
<code html>
  <a target="_blank" href="http://www.w3.org/">Link in einem neuen Fenster öffnen</a>
</code>

Würden Sie in diesem Beispiel den Linktext W3C aktivieren, würde die Zieladresse (hier: http://www.w3.org) hier tatsächlich in einem neuen Fenster oder Tab geöffnet und geladen werden. Ziel der Verwendung von target="_blank" ist natürlich vorrangig, den

===== E-Mail-Links =====

E-Mail-Verweise beginnen bei <color #ed1c24>href="mailto:EMAIL-ADRESSE"</color>. Damit wird die E‐Mail-Anwendung mit einer bestimmten E-Mail-Adresse geöffnet.

<code html>
  <a href="mailto:hans.muster@beispiel.ch">E-Mail senden</a> 
</code>

===== Telefon-Links =====

Telefon-Verweise beginnen bei <color #ed1c24>href="tel:NUMMER"</color>. Damit wird die Nummer gewählt und angerufen. (Desktop z.B. Facetime oder Online-Telefonie)

<code html>
  <a href="tel:044 123 45 67">E-Mail senden</a> 
</code>

===== Downloadlinks =====

Das Attribut download ermöglicht Links als Downloadverweis hinzuzufügen und das unabhängig vom Inhaltstyp (MIME-Typ).

<code html>
  <a href="dokument.pdf" download>Dokument herunterladen</a>
</code>

===== Anker - Links zu bestimmten Teilen einer Webseite setzen =====

Nichts kann für den Besucher lästiger sein, als eine sehr lange wissenschaftliche Abhandlung eines speziellen Themas auf einer Webseite zu lesen und dabei sehr lange hoch- und herunterscrollen zu müssen, um zu einem speziellen Abschnitt zu gelangen. Für solche Fälle können Sie sogenannte Anker mit dem globalen Attribut id setzen, den Sie dann mit einem gewöhnlichen Link mit dem a-Element anspringen können. Vorbildlich werden solche Zielanker z. B. bei Wikipedia für das Inhaltsver- zeichnis eines Themas verwendet.
Für eine Verlinkung zu einem bestimmten Bereich einer Webseite benötigen Sie nur:

  * einen Anker (Sprungmarke), den Sie mit dem Attribut <color #ed1c24>id="ankername"</color> erstellen. Zum Beispiel: <color #ed1c24><h1 id="ankername">Überschrift xyz</h1></color>
  * einen Link, der auf den Anker mit <color #ed1c24>href="#ankername"</color> verweist. Hierzu wird das Doppelkreuz-Zeichen <color #ed1c24>#</color> vor den Ankernamen geschrieben. Zum Beispiel: <color #ed1c24><a href="#ankername">Zur Überschrift xyz springen</a></color>
