====== LU07.A04 - OneDrive: Backup, Versionierun, Berechtigungen ======

===== Rahmenbedingungen =====
  * Sozialform: Einzelarbeit
  * Hilfsmittel:
  * Zeit: 15 Minutes
  * Resultat: 
    * Ein PDF das Ihre Erkenntnisse gemäss Fragestellung beantwortet. 
    * Dateien wurden Versioniert und gelöschte Dateien konnten wieder hergestellt werden. 

===== Ausgangslage ======
Sie haben erfahren, dass oneDrive eine sogenannte Backup-Funktion inkl. Recovery hat. Zudem wird jede Datei versioniert. Zudem können Sie Datei und Ordner mit anderen "teilen". Wenn man alle drei Features beherrscht, schläft man besser, da keine Datenverluste oder ungewollte Zugriff möglich ist.

===== Auftrag =====
  - Versuchen Sie herauszufinden, wie Sie erfahren können, welche Versionen von Dateien aktuell vorhanden sind.
  - Versuchen Sie eine alter Version dieser Datei wieder herzustellen.
  - Löschen Sie eine überflüssige Datei absichtlich.
  - Versuchen  Sie nun diese Datei Sie aus der Cloud wieder herzustellen.
  - Teilen Sie eine Datei mit einer zwei anderen Mitlernenden und versuchen Sie gemeinsame diese Datei zu bearbeiten.
  - Halten Sie Ihre Erkenntnisse schriftlich nachfollziehbar fest.
