==== LU18.A04 Projekt beantragen ====

Überlegen Sie sich ein kleines Projekt (Umfang ca. gleich wie bei den bisherigen vorgegebenen Projekten).

Vorgaben (bei diesem spezifischen Projekt):
  * Einzelarbeit
  * Programmiersprache: Javascript/Typescript (zusätzlich Java oder Python als Ergänzung möglich)
  * Das Projekt muss ein Web-Frontend beinhalten und dadurch eine standalone-Webseite "publizieren" können. -> Dies schliesst z.B. Browser-Plugins aus. 

Beispiele:
  * URL-Shortener
  * QR-Code generator
  * Terminumfrage-Tool
  * ...

Reichen Sie Ihr Projekt mit dem Projektformular ein.
