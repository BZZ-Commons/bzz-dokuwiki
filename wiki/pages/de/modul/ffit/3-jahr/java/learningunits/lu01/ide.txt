====== LU01c - IDE ======

Installieren Sie eine IDE ihrer Wahl für Java-Programmierungen.

**Intellij IDEA**  \\
Jetbrain-Produkt für Java and Kotlin
  * https://www.jetbrains.com/de-de/idea/download/

**Eclipse**  \\
Auf Java spezialisierte IDE
  * https://www.eclipse.org/downloads/

**VS Code**  \\
Vielseitig benutzbar
  * https://code.visualstudio.com/download

**Weitere IDEs (NetBeans, ...)**   \\
Auf eigene Verantwortung

Passen Sie bei Bedarf die IDE nach Ihren Vorlieben an.
