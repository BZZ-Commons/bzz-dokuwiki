==== LU11.A02 pgvector installieren====

Wie es der Zufall so will, gibt es eine Erweiterung für PostgreSQL names ''pgvector'', die eine Vektordatenbank ermöglicht.

Installieren Sie die Erweiterung für PostgreSQL 17 folgendermassen:
  * https://github.com/andreiramani/pgvector_pgsql_windows/releases/tag/0.8.1_17.6 runterladen und readme.txt lesen
  * Im korrekten Verzeichnis entpacken
  * <code bash>psql -U postgres</code>
  * <code psql>CREATE EXTENSION vector;</code>
  * <code psql>SELECT extname,extrelocatable,extversion FROM pg_extension where extname='vector';</code>
  * <WRAP><code psql>CREATE TABLE documents (
    id SERIAL PRIMARY KEY,
    content TEXT,
    embedding vector(384) -- number of dimensions
);</code></WRAP>
