====== LU01a - AWS Build-Environment ======

==== Grundidee ====
Es gibt eine zentrale Build Environment für die gesamte Doppelklasse. Dadurch sind Gruppenarbeiten möglich und es gibt eine Gesamtübersicht über die gesamte Klasse.

Um die Build-Environment (Instanzen, Container etc.) müssen Sie sich selber nicht kümmern. Sie können sie einfach nutzen.

==== Zugriff ====
IP-Adresse und Zugangsdaten in Teams.

==== Ports ====
^ Port ^ Beschreibung ^
| 80 | Standard bei HTTP im Browser |
| 8080 | Alternativer HTTP-Port. Wird von vielen für Admin-Oberfläche verwendet |
| 9000 | Standard-Port bei SonarQube |
| 50000 | Port für Jenkins Build Agents |
