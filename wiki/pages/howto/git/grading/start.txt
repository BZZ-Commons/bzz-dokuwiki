====== Resultate von Github-Actions in Moodle übernehmen ======
<WRAP center round info 60%>
Mit dem automatischen Testen von Abgaben im Classroom erhalten Lernende und Lehrpersonen direkt ein Feedback zum abgegebenen Code.
Um als Lehrperson eine Übersicht über die Leistungen eines Lernenden zu erhalten, muss du jedoch jedes einzelne Assignment einzeln öffnen.

Diese Anleitung beschreibt, wie die Bewertung der Tests aus GitHub in Moodle übernommen werden.
</WRAP>
