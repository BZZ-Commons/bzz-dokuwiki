====== GitHub Classroom - Assignment ======
<WRAP center round info 60%>
Die Programmieraufgaben sind als Assignment im GitHub Classroom vorbereitet.
Wenn Du das Assignment im Classroom akzeptierst, wird ein persönliches Repository auf GitHub erstellt.
</WRAP>

===== Vorgehen =====
Von deiner Lehrperson erhältst du jeweils einen Link zur Aufgabe (Assignment) im GitHub Classroom.

  - Bevor du den Link öffnest, melde dich auf [[https://github.com]] mit deinem GitHub-Account für das BZZ an.
  - Öffne nun den Link im Webbrowser.

=== Identifizieren ===
Bei der ersten Aufgabe in einem neuen Classroom, musst du dich zuerst identifizieren:

{{:howto:git:classroom_student_00.png?400|}}

  * Wähle aus der Liste deinen Namen aus und bestätige deine Auswahl.
  * Die Namensliste wird nicht angezeigt? Dann bist du schon im Classroom registiert.
  * Dein Name steht nicht in der Liste? Dann schreibe eine Nachricht an deine Lehrperson und klicke auf "Skip to the next step" .

=== Assignment akzeptieren ===
{{:howto:git:classroom_student_01.png?400|}}

Mit einem Klick auf [Accept Assignment] nimmst du die Aufgabe an. 
GitHub erstellt nun eine persönliche Kopie des Repositories für dich. 
Dies kann einige Sekunden dauern. 
Warte etwa 30 Sekunden und lade die Webseite neu, bis dir der Link zu deinem Repository angezeigt wird.

{{:howto:git:classroom_student_02.png?400|}}

Klicke auf diesen Link um zu deinem Repository zu gelangen.

{{:howto:git:classroom_student_03.png?400|}}
