====== GitHub Benutzernamen in Moodle eintragen ======
Um die Testresultate aus GitHub Classroom ins Moodle zu übertragen, müssen Sie Ihren GitHub-Benutzernamen im Moodle-Profil eintragen.

1. Melden Sie sich in Moodle an (https://moodle.bzz.ch)

2. Öffnen Sie Ihr Benutzerprofil

{{:howto:git:github_username_01.png?300|}}

3. Wählen Sie "Profil bearbeiten"

{{:howto:git:github_username_02.png?400|}}

4. Zuunterst finden Sie den Bereich "GitHub Classroom". Tragen Sie hier Ihren GitHub-Benutzernamen ein.

{{:howto:git:github_username_03.png?400|}}

5. Klicken Sie auf [Profil aktualisieren] um die Änderung zu speichern.
