====== LU02.L04 - Multiplikation und Division ======

{{:modul:m319:learningunits:lu02:loesungen:lu02_l04_multiplikation.png?800|}}

===== Python =====
<code python>
number1 = float(input('Erste Zahl:'))
number2 = float(input('Zweite Zahl:'))
print(str(number1) + ' * ' + str(number2) + ' = ' + str(number1 * number2))
print('Durchschnitt von ' + str(number1) + ' und ' + str(number2) + ' = ' + str((number1 + number2) / 2))
</code>
