====== LU03.A03 - Message======
<WRAP center round todo 60%>
Schreiben Sie ein Programm das ein Echo der Eingabe ausgibt.
</WRAP>

===== Auftrag =====

Schreiben Sie ein Programm, das den Benutzer mit **"Write a message..."** auffordert, eine Zeichenkette zu schreiben. Wenn der Benutzer eine Zeichenkette eingegeben hat (d.h. einen Text geschrieben und die Eingabetaste gedrückt hat), sollte das Programm die vom Benutzer eingegebene Zeichenkette ausgeben.

Fügen Sie folgende Programmvorlage in Ihre IDE ein:

<code python>
def main():
    # Write your code here

if __name__ == '__main__':
    main()
</code>

===== Programmablauf =====

Beispiel für die Ausgabe, wenn der Benutzer "Bye" schreibt.

<code>
Write a message...
*Bye*
Bye
</code>

Beispiel für die Ausgabe, wenn der Benutzer schreibt "Once upon a time...".

<code>
Write a message...
*Once upon a time...*
Once upon a time...
</code>


Anmerkung: Kümmern Sie sich im Moment nicht zu sehr um ''if %%__%%name%%__%% == '%%__%%main%%__'%%:''. Wir brauchen es technisch gesehen nicht für dieses Programm, aber es ist eine gute Übung, es einzubauen, und es wird in späteren Übungen klarer werden.
