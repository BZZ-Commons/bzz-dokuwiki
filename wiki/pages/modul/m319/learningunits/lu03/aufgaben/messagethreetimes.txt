====== LU03.A05 - Message three times ======
<WRAP center round todo 60%>
Schreiben Sie ein Programm, das ein drefaches Echo ausgibt.
</WRAP>


===== Auftrag =====

Fügen Sie folgende Programmvorlage in Ihre IDE ein:

<code python>
def message_three_times():
    # Write your code here

if __name__ == '__main__':
    message_three_times()
</code>

Schreibe ein Programm, das den Benutzer mit **"Write a message..."** auffordert, eine Zeichenkette zu schreiben. Wenn der Benutzer eine Zeichenkette eingegeben hat (d.h. einen Text geschrieben und die Eingabetaste gedrückt hat), muss das Programm die Zeichenkette des Benutzers **dreimal** ausdrucken.

===== Programmablauf =====
<HTML>
<iframe src="https://trinket.io/embed/python3/3f8b920202?outputOnly=true&start=result" width="100%" height="200" frameborder="0" marginwidth="0" marginheight="0" allowfullscreen></iframe>

Beispiel für die Ausgabe, wenn der Benutzer die Zeichenfolge "Hi" schreibt.
<code>
Write a message:
*Hi*
Hi
Hi
Hi
</code>

Beispiel für die Ausgabe, wenn der Benutzer schreibt "Once upon a time...".

<code>
Write a message:
*Once upon a time...*
Once upon a time...
Once upon a time...
Once upon a time...
</code>


Anmerkung: Kümmern Sie sich im Moment nicht zu sehr um ''if %%__%%name%%__%% == '%%__%%main%%__'%%:''. Wir brauchen es technisch gesehen nicht für dieses Programm, aber es ist eine gute Übung, es einzubauen, und es wird in späteren Übungen klarer werden.
