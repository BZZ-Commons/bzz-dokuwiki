====== LU09.A04 - Quersumme ======

<WRAP center round todo 60%>
Schreibe ein Programm, das den Benutzer nach einer ganzen Zahl fragt und die Quersumme dieser Zahl berechnet und ausgibt.
</WRAP>

===== Auftrag =====
**Hinweis:** Variable Elemente der Ausgabe sind speziell ''formatiert''

Erstelle die Programmlogik für das Programm:
  - Das Programm fordert den Benutzer auf, eine Zahl einzugeben.
  - Das Programm berechnet dann die Quersumme der eingegebenen Zahl.
  - Das Programm gibt die Quersumme aus: "Die Quersumme ist: ''summe''".

==== Beispiele ====

<code>
Bitte eine Ganzzahl eingeben für Quersummenberechnung:
*12345*
Die Quersumme ist: 15
</code>
<code>
Bitte eine Ganzzahl eingeben für Quersummenberechnung:
*101010*
Die Quersumme ist: 3
</code>
<code>
Bitte eine Ganzzahl eingeben für Quersummenberechnung:
*7*
Die Quersumme ist: 7
</code>

===== Vorgehen =====

  - Akzeptiere das GitHub Classroom Assignment im Moodlekurs.
  - Klone das Repository in PyCharm.
  - Codiere die Programmlogik in ''digit_sum.py''.
  - Teste dein Programm mit den Testfällen in ''digit_sum_test.py''.
  - Führe einen Commit und einen Push durch.

=== Abgabe ===
Die Abgabe erfolgt durch den Push ins GitHub Repository.
In Moodle ist keine Abgabe vorgesehen oder möglich.
