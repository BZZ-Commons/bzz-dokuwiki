====== LU05.A04: Odd or Even ======
<WRAP center round info 60%>
Schreibe ein Programm, das den Benutzer nach einer Zahl fragt und uns mitteilt, ob sie gerade oder ungerade ist.
</WRAP>

===== Auftrag =====
Erstelle die Programmlogik für das Programm:
  - Das Programm fordert den Benutzer auf, eine Zahl ''number'' einzugeben
  - Falls die Zahl gerade ist, gibt das Programm ''Number {number} is even.'' aus.
  - Sonst gibt das Programm ''Number {number} is odd.'' aus.

==== Beispiel ====

<code>
Give a number:
*2*
Number 2 is even.
</code>
<code>
Give a number:
*7*
Number 7 is odd.
</code>

===== Vorgehen =====

  - Akzeptiere das GitHub Classroom Assignment im Moodlekurs.
  - Klone das Repository in PyCharm.
  - Codiere die Programmlogik in ''positivity.py''.
  - Teste dein Programm mit den Testfällen in ''positivity_test.py''.
  - Führe einen Commit und einen Push durch.

=== Abgabe ===
Die Abgabe erfolgt durch den Push ins GitHub Repository.
