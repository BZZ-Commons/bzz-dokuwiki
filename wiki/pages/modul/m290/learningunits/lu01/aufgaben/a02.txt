===== Introduction  =====
Programming source code is not possible without a propper code editor. As it happened, Webstorm is one such editor that offers us a variety of useful features, and fortunately it is free for educational purposes. Doesn't that sound great!

===== Assignments =====
  - If have not already done, create an jetbrains account
  - Download and install the code editor Webstorm, which is free for students: [[https://www.jetbrains.com/webstorm/| Webstorm/Download]]
  - Carry out the post installation step by installing the database module plug in in webstorm 
  - Establish the connection between database and webstorm
  - Check your installation using the following functions 
        * New project
        * New file
        * Rename a new file.
        * Find the stored file directory of webstorm
        * Establish a connection between Webstorm and Mysql
        * Select current databases from MySQL
