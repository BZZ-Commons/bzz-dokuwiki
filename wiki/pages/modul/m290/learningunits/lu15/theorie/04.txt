====== LU02b2 - LB2 - Lernziele ======
===== LU05: SQL | DLQ commands =====
  - Syntax einer typischen DQL-Anweisung kennen und erklären können.
  - Select auf einer Tabelle durchführen können
  - Select auf 2-3 Tabellen
  - Mit ohne ohne Alias
  - Mit und ohne Filter
  - Unterschiedliche Sortierkriterien und Sortierungen
  - Anwendung von Aggregat Funktionen

===== LU06: SQL | DDL commands =====
  - Syntax einer typischen DDL-Anweisung kennen und erklären können.
  - Anlegen, verwenden und löschen von DB-Schemata (Instanzen)
  - Anlegen, verändern und löschen von Tabellen
  - Anlegen und verändern von Zusatzbedingungen (Constraints) 

===== LU07: SQL | DML commands =====
  - Syntax einer typischen DML-Anweisung kennen und erklären können.
  - Einfügen von Datensätzen in Tabellen nach Vorgabe
  - Verändern von Tabelleninhalten
  - Verändern von Tabelleninhalten nach Vorgaben
  - Löschen von Tabelleninhalten nach Vorgaben
