====== LU04a - W3School Tutorial's Overview  ======

===== Learning Objectives =====
  - Know how to access to the learning material.
  - Identification of the structure/chapters of the learning material.
  - Know which functionalities the W3School tutorial offers.

===== Overview =====
The W3School provides us with a series of useful tutorials that are both well structured and free to access. In detail, each section of the topic in question consists of three key elements, as shown in the screenshot below:

  - Sections of expertise we want to achieve, such as SQL or JavaScript.
  - Available SQL function including the general syntax, a short description and some practical examples.
  - A sandbox for testing some SQL commands, provided for instant practising the examples without having to install a lot of software.
  - SQL editor with ready-to-use table structure filled with data
  - Exercises area with useful questions
  - Quiz area to make round our new SQL skills
