===== Inhalte =====
Bei dieser Aufgabe geht es primär um die HTML-Tags und nicht um eine umfangreiche Zusammenfassung.
Erstellen Sie pro Learning Unit:
  * Kurze, stichwortartige Zusammenfassung
  * Ein Code-Beispiel aus einer der Aufgaben \\ (lesen Sie [[https://wiki.selfhtml.org/wiki/HTML/Tutorials/Code_in_HTML_darstellen|hier]] wie Sie code in HTML darstellen können).

==== Beispiel ====
**HTML** = HyperText Markup Language, Inhalt und Struktur der Webseite \\
**CSS** = Cascading Style Sheet, Darstellung und Optik

<code html>
<!DOCTYPE html>
<html lang="de">
  <head>
    <meta charset="UTF-8">
    <title>Demo - Selection</title>
  </head>
  <body>
  ...
  </body>
</html>
</code>

  - Validieren Sie Ihren HTML-Code auf Einhaltung der Standards.
  - Geben Sie Ihre Lösung mittels Push ins Repository ab.
