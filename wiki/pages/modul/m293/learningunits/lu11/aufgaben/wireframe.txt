====== LU11.A02 - Wireframe erstellen ======

<WRAP center round todo 60%>
Erstellen Sie zwei Wireframes für den Fussballclub "1 FC Hinterforst". Ein Wireframe soll dabei eine Mobile-Ansicht zeigen, der andere die Desktop ansicht.
</WRAP>

In Aufgabe [[modul:m293:learningunits:lu08:aufgaben:fussballwebsite|LU08.A02]] haben Sie eine Website für den 1 FC Hinterforst erstellt. Dieser Fussballclub möchte seine Webpräsenz in einem neuen Kleid erscheinen lassen. Nachdem Sie in der [[modul:m293:learningunits:lu11:aufgaben:styletile|letzten Aufgabe]] einen Style Tile erstellt haben, ergänzen Sie nun das Layout durch zwei Wireframes.

===== Vorgaben =====
  * Das Tool ist ihnen freigestellt, Tools wie Balsamiq, Stift Papier oder auch Powerpoint können gut verwendet werden.
  * Als Resultat soll ein PDF auf Moodle geladen werden.
