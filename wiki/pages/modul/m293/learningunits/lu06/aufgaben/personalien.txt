====== LU06.A01 - Personalien ======
<WRAP center round todo 60%>
Erstellen Sie eine HTML-Seite mit einem Formular zum Erfassen der Personalien.
</WRAP>

===== Vorgehen =====
  - Akzeptieren Sie das GitHub Classroom Assignment für die LU06.
  - Klonen Sie das Repository.
  - Erstellen Sie die HTML-Webseite in der Datei ''personal.html''.
  - Validieren Sie Ihren HTML-Code auf Einhaltung der Standards.
  - Geben Sie Ihre Lösung mittels Push ins Repository ab.

===== Formular =====

Ihr Formular muss folgende Vorgaben einhalten:
  * Zieladresse: https://it.bzz.ch/demo/formdata/index.php
  * Eingabefelder:
    * Vorname
    * Nachname
    * Wohnort
  * Buttons:
    * Absenden
    * Zurücksetzen

===== Testen =====
Sie können Ihre Formular testen, indem Sie die Daten absenden.
Die Antwort des Servers muss wie folgt aussehen:

<code>
HTTP-Methode:   POST
Eingabefelder:
  givenname  =  a
  surname    =  b
  residence   =  c
</code>
