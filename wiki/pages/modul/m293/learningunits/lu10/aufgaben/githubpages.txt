====== LU10.A02 - GitHub Pages ======
<WRAP center round info 60%>
Veröffentlichen Sie eine Webseite mit GitHub Pages
</WRAP>

===== Vorgehen =====
  - Öffnen Sie Ihr persönliches Repository der Aufgabe [[modul/m293/learningunits/lu09/aufgaben/webstorm]] auf GitHub.
  - Richten Sie GitHub Pages ein.

Führen Sie einige Änderungen an Ihrem HTML-Code durch und führen Sie einen Push ins GitHub-Repository durch. Nachdem der Workflow ''pages build and deployment" fertig ist, ist Ihre aktualisierte Webseite verfügbar.
