====== LU04.A01 - Speichergrössen ======

===== Ausgangslage =====
Sie haben sich über die Grundlagen der menschlichen Informationsverarbeitung informiert. Nun wollen wir in Erfahrung bringen, welche Datenmengen unser Gedächtnis verarbeiten kann. Der Speicherbedarf für die menschlichen Wahrnehmungen hängt von seinem Sinnesorgan ab: Lesen benötigt weniger Speicher im Gedächtnis als das Sehen. In einer Untersuchung wurden die Zahlen in der ersten Spalte ermittelt:

^ ^ pro Stunde ^ pro Tag ^ pro Jahr ^ im Leben ^
| Lesen | 50 KB | | | |
| Hören | 50 MB | | | |
| Sehen | 1 GB | | | |

===== Aufgabenstellung =====

Ergänzen Sie die Tabelle, um den Speicherbedarf eines Menschen nach 80
Lebensjahren zu ermitteln, wobei der Mensch pro Tag nur folgende „Leistungen“
erbringen kann:
  * Lesen: 8 Stunden/Tag.
  * Hören in MP3/CD-Qualität: 16 Stunden/Tag
  * Sehen in Fernsehqualität: 20 Stunden/Tag
