===Übung 4===
**Aufagbe 1**\\
Gegeben ist folgender Code
<code python>
value_A = True
value_B = False
value_C = True

x = not value_A or value_B and value_C

print(x)
</code>
Welche Ausgabe (Wert von x) wird angezeigt?\\
\\
**Aufgabe 2**\\
Gegeben ist folgender Code
<code python>
x = value_A and value_B or value_C
</code>
Unter welchen Bedingungen wird ''x'' den Wert ''True'' haben?\\
