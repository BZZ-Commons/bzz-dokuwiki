====== LU05.L03 - Funktionale Anforderungen ======

Folgende Anforderungen sind Beispiele für funktionale Anforderungen:

  * **Benutzeranmeldung:** Das System muss es Benutzern ermöglichen, sich mit einem Benutzernamen und Passwort anzumelden.
  * **Produktsuche:** Das System muss eine Suchfunktion bieten, mit der Benutzer Produkte nach Name, Kategorie oder Hersteller suchen können.
  * **Warenkorb:** Das System muss einen Warenkorb bieten, in den Benutzer Produkte für den späteren Kauf legen können.
  * **Bestellvorgang:** Das System muss den Bestellvorgang von der Auswahl bis zur Zahlung unterstützen.
  * **Rückmeldung:** Das System muss dem Benutzer nach erfolgreich abgeschlossener Bestellung eine Bestätigungs-E-Mail senden.
