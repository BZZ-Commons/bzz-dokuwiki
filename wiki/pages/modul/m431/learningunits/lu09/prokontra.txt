====== LU09b - Pro und Kontra ======
<WRAP center round info 60%>
Bei dieser Methode werden zu jeder Lösungsvarianten die Vorteile (Pro) und Nachteile (Kontra) aufgelistet.
</WRAP>

Die Pro-Kontra-List ist denkbar einfach.
Man nimmt ein Blatt und zieht einen senkrechten Strich in der Mitte.
Auf der linken Seite notiert man alle Punkte bzw. Argumente die für diese Lösungsvariante spricht.
Alle Argumente, die gegen die Lösungsvariante sprechen, notiert man auf der rechten Seite.

Schon die Anzahl der Argumente auf der Pro- bzw. Kontra-Seite sind ein Indiz, welche Entscheidung man fällen sollte.
Dennoch macht es Sinn, die verschiedenen Argumente zu gewichten.
Möglicherweise wiegt ein Kontra-Argument so schwer, dass es mehrere Pro-Argumente aufwiegt.

===== Beispiel =====
Soll ich mit 3 Freunden eine Wohngemeinschaft gründen?

^ Pro ^ Kontra ^
| ++ Geringere Kosten für Miete, Strom, ...  | - Streit um Regeln, Budget  |
| + Gemeinschaft  | -- Wenig Privatsphäre  |
| + Einrichtungskosten teilen  |   |
| + Gegenseitige Hilfe  |   |

In diesem Beispiel wurde eine einfache Gewichtung vorgenommen:
Je mehr "+" oder "-" vor einem Argument stehen, desto wichtiger ist es für mich.
Natürlich könnte man die Gewichtung noch weiter ausbauen und z.B. eine Punkteskala von 0=nicht wichtig bis 5=sehr wichtig verwenden.
Damit wird aber die Einfachheit der Pro/Kontra-Liste weitgehend eliminiert.
