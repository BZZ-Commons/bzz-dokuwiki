====Auftrag 14: User Story Map erstellen====
Übersetzen Sie nun die Szenarien in User Stroy Maps.
 
===1. User Story Map erstellen===
{|style = "table-layout: fixed; width: 100%;"
|- 
|style = "width: 100px; border-top: thin solid; vertical-align:top;"| Sozialform:
|style = "width: 100%"   | Gruppenarbeit, 4-5 Lernende\\ Teilen Sie sich in 2 Gruppen auf, um je eine User Story Map zu erstellen.
|-
|style="border-top: thin solid; "| Ergebnis:   
| Eine ausgearbeitet User Story Map je Szenario.
|-
|style="border-top: thin solid; vertical-align: top;"| Vorgehen:   
| Halten Sie die Teilaufgaben aus dem Szenario stichwortartig fest und legen Sie dann einen Namen für die übergeordnete Aufgabe fest.\\
Überlegen Sie sich zu jeder Teilaufgabe die möglichen User Stories.\\
Erstellen Sie dann aus den vorliegenden Artefakte die Map.
|-
|style="border-top: thin solid; vertical-align: top;"| Dauer:      
| 30 Minuten
|-
|style="border-top: thin solid; vertical-align: top;"| Artefakte:  
| Zwei User Story Maps im Miro-Board.
