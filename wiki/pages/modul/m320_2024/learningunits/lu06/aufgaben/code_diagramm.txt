====== LU06.A02: Sequenzdiagramm zu Code zeichnen ======

<WRAP center round todo 60%>
Sie können aus einem einfachen Programcode das zugehörige Sequenzdiagramm skizzieren.
</WRAP>

===== Auftrag =====
Studieren Sie den Code der Klasse MiniChat (aus [[.:kommunikation]]) in seiner Grundform (mit den Objekten ''max'', ''moritz'' und ''lempel''). 
Sikzzieren Sie das dazu passende Sequenzdiagramm auf.

=== Hinweis: === 
Für Sequenzdiagramme bietet Ihnen MS-Visio ein entsprechendes Shape an.

===== Dauer =====
10 - 15 Minuten

===== Abgabe =====
Geben Sie das Dokument als PDF-Datei in Moodle ab.
