====== LU06.A01: Objekte kommunizieren ======

<WRAP center round todo 60%>
Sie können durch Methodenaufrufe Daten zwischen Objekten austauschen.
</WRAP>


===== Auftrag ===== 
  - Akzeptieren Sie das Assignment im GitHub Classroom und klonen Sie Ihr Repository in die Entwicklungsumgebung.
  - Führen Sie die ''main''-Routine aus und überlegen Sie sich, wie die Kommunikation zwischen den Objekten abläuft.
  - Ändern Sie die Kommunikation, in dem Sie in ''main'' eigene Objekte erstellen und einen Dialog kreieren. Wichtig dabei: den Namen der Objekte müssen Sie immer über den getter ''name'' lesen.

===== Dauer =====
10 Minuten

===== Abgabe =====
Führen Sie einen Push Ihres Repositories durch.
