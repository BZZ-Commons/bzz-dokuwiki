====== Übungen: Github Assignment lösen ======

===== Github-Classroom beitreten =====

Sie erhalten von Ihrem Lehrer eine Einladung um einem Github-Classroom beizutreten.
Folgen Sie dem Link und wählen Sie Ihren Namen aus der Liste aus: 

{{ :modul:m319:learningunits:lu06:aufgaben:joinclassroom.png?800 |}}

<WRAP center round tip 60%>
Dadurch wird Ihr Github-Username mit Ihrem richtigen Namen verknüpft. Das erleichtert der Lehrperson Ihre Abgaben zu identifizieren. Diese Auswahl des Namens müssen Sie nur einmal pro Classroom (pro Modul) machen.
</WRAP>

===== Assignment annehmen =====
Sobald Sie dem Classroom beigetreten sind, können Sie das Assignment (die Aufgabe) annehmen.

{{ :modul:m319:learningunits:lu06:aufgaben:acceptassignment.png?800 |}}

Github erstellt ihnen nun ein eigenes Repository für ihr Assignment, das dauert einige Sekunden. Aktualisieren Sie die Webseite von Zeit zu Zeit.

{{ :modul:m319:learningunits:lu06:aufgaben:acceptedassignment.png?800 |}}

Sobald das Repository bereit ist, können Sie es mit dem Link aufrufen.

{{ :modul:m319:learningunits:lu06:aufgaben:readytogo.png?800 |}}

Ihr persönliches Repository ist nun bereit.

{{ :modul:m319:learningunits:lu06:aufgaben:repoready.png?800 |}}

===== Aufgaben =====

  * Klonen Sie das Repository
  * Lösen Sie die TODO's im File Bootsliste.java
