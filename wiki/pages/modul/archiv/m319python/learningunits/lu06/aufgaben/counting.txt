====== LU06.A10: Counting ======

Schreiben Sie ein Programm, das eine ganze Zahl vom Benutzer liest. Anschließend gibt das Programm Zahlen von 0 bis zu der vom Benutzer angegebenen Zahl aus. Sie können davon ausgehen, dass der Benutzer immer eine positive Zahl angibt. Im Folgenden finden Sie einige Beispiele für die erwartete Funktionalität.
<code>
**4**
0
1
2
3
4
</code>
<code>
**1**
0
1
</code>

<WRAP center round important 60%>
Implementieren Sie die Aufgabe mit einer ''for'' und einer ''while''-Schleife
</WRAP>


===== Vorlage =====
<code python>
def main():
  # Write your program here

if __name__ == '__main__':
    main()
</code>

Anmerkung: Kümmern Sie sich im Moment nicht zu sehr um ''if %%__%%name%%__%% == '%%__%%main%%__'%%:''. Wir brauchen es technisch gesehen nicht für dieses Programm, aber es ist eine gute Übung, es einzubauen, und es wird in späteren Übungen klarer werden.
