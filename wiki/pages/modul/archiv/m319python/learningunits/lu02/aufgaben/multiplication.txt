====== LU02.A06: Multiplication formula ======

===== Auftrag =====

Erstellen Sie, ähnlich wie in der vorherigen Übung, ein Programm, das die in zwei Integer-Variablen gespeicherten Werte multipliziert.

Wenn die eingegebenen Zahlen zum Beispiel 2 und 8 sind, sollte das Programm Folgendes ausgeben:

Beispielausgabe:

<code>
Give the first number:
*2*
Give the second number:
*8*
2 * 8 = 16
</code>

Wenn die eingegebenen Zahlen 277 und 111 sind, sollte der Ausdruck wie folgt aussehen:

<code>
Give the first number:
*277*
Give the second number:
*111*
277 * 111 = 30747
</code>


Verwenden Sie folgende Programmiervorlage:

<code python>
def main():
    #write your code below this line

if __name__ == '__main__':
    main()
</code>


Anmerkung: Kümmern Sie sich im Moment nicht zu sehr um ''if %%__%%name%%__%% == '%%__%%main%%__'%%:''. Wir brauchen es technisch gesehen nicht für dieses Programm, aber es ist eine gute Übung, es einzubauen, und es wird in späteren Übungen klarer werden.
