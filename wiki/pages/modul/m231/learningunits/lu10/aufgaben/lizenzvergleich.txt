====== LU10.A01 - Lizenzmodelle ======

<WRAP center round todo 60%>

Ich nehme was, mache was und gebe es weiter. Darf ich das?

</WRAP>

==== Ausgangslage ====

Um eine Software zu schreiben, benötigen Sie zuerst wiederum eine oder mehrere Tools. Diese stehen unter irgend einer Form von Lizenz. Diese Lizenz(en) können wiederum Regeln für die Weitergabe Ihrer Software bedeuten. Aber welche...?

==== Auftrag ====
Schauen Sie sich mal von nachstehenden Tools die Lizenzbestimmungen an, insbesondere Bestimmungen zu KOPIEREN, VERÄNDERN und VERBREITEN. Beschränken Sie sich dabei auf die aktuellste Version der Tools und deren Bestimmungen.

  * Java JDK (Oracle)
  * OpenJDK
  * Python
  * Flutter
  * Microsoft Visual Studio

Dokumentieren Sie, unter welcher Lizenzart das Tool selber steht und unter welchen Bedingungen Sie Ihre eigene damit erstellt Software verbreiten können. Zum Beispiel stellt sich die Frage der kommerziellen Nutzung.

Stellen Sie Ihre Erkenntnisse strukturiert und übersichtlich zusammen.

Geben Sie Ihre Dokumentation als PDF ab
