====== LU07.A01 - Authenticator einrichten ======

<WRAP center round todo 60%>
Installieren Sie eine Authenticator-App auf Ihrem Smartphone.
Richten Sie eine 2-Faktor-Authentifizierung für eine Webseite ein.
</WRAP>


==== Mögliche Webseiten ====
Dies sind einige Webseiten, die eine 2FA anbieten:
  * Digitec/Galaxus
  * GitHub
  * Office 365
  * Google

===== Abgabe =====
Schreiben Sie im Textfeld der Moodle-Aufgabe, welchen Authenticator/Webseite(n) Sie ausprobiert haben.


Sie können aber auch eine beliebige andere Webseite wählen.
