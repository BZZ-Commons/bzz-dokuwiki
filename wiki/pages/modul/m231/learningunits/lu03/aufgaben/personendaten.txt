====== LU03.A01 - Personendaten suchen ======

<WRAP center round todo 60%>
Erstellen Sie eine Statistik von Personendaten auf Ihren Geräten und sozialen Medien.
Tauschen Sie sich über Ihre Ergebnisse mit Ihrem Gruppenpartner aus.
</WRAP>

Meistens haben wir kaum einen Überblick über all die Daten, welche wir speichern und im Web veröffentlichen.
  * Haben Sie sich schon mal gefragt, wieviele dieser Daten einen Bezug zu einer Person haben?
  * Wie schätzen Sie den möglichen Schaden ein, falls diese Daten veröffentlicht werden?

==== Beispiele ====
  * Mein Testament in dem ich meine Familie enterbe.  \\ => Zumindest peinlich, da es auf familiäre Probleme hinweist.
  * Mitgliederliste des Unihockeyvereins mit Adresse, Geburtsdatum, Telefonnummern. \\ => Eher harmlos.
  * Urlaubsfotos von meinen Angelferien in Finnland. \\ => Vermutlich harmlos.
  * Video der besten Freundin bei einer wilden Party. \\ => Wurde für die ehemalige finnische Ministerpräsidentin zur Katastrophe.

===== Vorlage Statistik =====
Nehmen Sie sich je 10 Minuten Zeit für Ihren Laptop, Ihr Smartphone und Ihre Social Media-Seiten.
Suchen Sie nach Bildern, Adressen, Emails, ... die Angaben zu einer oder mehreren Personen enthalten.

^ Person ^^ Einschätzung Schaden ^^^ 
^ Ich ^^ Harmlos ^ Peinlich ^ Katastrophe ^
| | Laptop | | | |
| | Smartphone | | | |
| | Social Media | | | |
^ Drittpersonen ^^ Harmlos ^ Peinlich ^ Katastrophe ^
| | Laptop | | | |
| | Smartphone | | | |
| | Social Media | | | |

Sie erhalten hier eine Excel-Vorlage zum ausfüllen:

===== Diskussion (10 Minuten) =====
Tauschen Sie sich mit Ihren Gruppenpartner über die Statistik aus.
Diskutieren Sie nicht über die Einzelheiten, sondern lediglich über die gefundene Menge and personenbezogenen Daten.
Mit welchen Massnahmen verhindern Sie, dass die peinlichen oder katastrophalen Daten nicht in die falschen Hände gelangen? 

===== Abgabe =====
Laden Sie die Statistik in der Moodle-Aufgabe hoch.
