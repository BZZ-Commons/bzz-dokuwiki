======= LU04.A01 - Kleinprojekt umsetzen ======

In dieser Aufgabe soll eine vordesignte Webseite anhand eines vorgegebenen Layouts umgesetzt werden. Dabei wird erwartet, dass die Seite mithilfe von HTML und CSS nach den Designvorgaben erstellt wird. Die exakte Darstellung der Seite sowie eine saubere Programmierung der Struktur stehen im Fokus.

Zunächst soll das Layout genau analysiert werden. Es ist auf verschiedene Bereiche wie Header, Navigation, Inhalte und Footer zu achten. Ebenso müssen die verwendeten Schriftarten, Farben und Abstände zwischen den Elementen beachtet werden.

Daraufhin wird die Struktur der Seite in HTML umgesetzt.
