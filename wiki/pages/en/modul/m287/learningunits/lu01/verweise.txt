====== LU01b - References and anchors ======

===== Learning objectives =====
  - I can add references (hyperlinks) to a web page and customise them using the attributes (e.g. href, id). HTML elements/tags are <a>.
  - I know the difference between absolute and relative references.
  - I can create links syntactically correctly and can list characteristics and properties for the correct structure.
  - I know the purpose of local references (anchors).
  - I can create local references (anchors) correctly.

===== Introduction =====

You can create a link in HTML with the <color #ed1c24><a></color>element (a = anchor) in HTML. The link text (or reference text) between the <a>-tags is activated if the href attribute has a valid value.

==== Example ====
A link text is usually underlined by the web browser (usually in blue). However, you can (as always) change this at any time using CSS.

<code><a href="http://www.bzz.ch">Bildungszentrum Zürichsee</a></code>

===== Inserting links to other HTML documents on your own website =====
The links connect the loose collections of HTML documents to form a coherent website. The so-called navigation of the website can be created from this. For a link to another page of the same website, it is sufficient to enter the relative (instead of the absolute) URL. More on this later.

==== Example ====

The following directory structure shown in the illustration is given as an example.

{{modul:m287:learningunits:lu01:02.png?600|}}

The link for the start page, here index.html, to the other pages <color #ed1c24>links.html</color>, <color #ed1c24>about.html</color> and <color #ed1c24>impressum.html</color> looks like this in practice:

<code html>
<!doctype html>
<html lang="de">
  <head>
    <title>Startseite - Blog</title>
    <meta charset="UTF-8">
  </head>
  <body>
    <nav>
      Blog |
      <a href="seiten/links.html">Links</a> |
      <a href="seiten/about.html">Über mich</a> |
      <a href="seiten/impressum.html">Impressum</a>
    </nav>
    <h1>Mein Blog</h1>
    <p>Neueste Berichte zu HTML</p>
  </body>
</html>
</code>

---

Of course, you must also include the links to the other pages, i.e. <color #ed1c24>links.html</color>, <color #ed1c24>about.html</color> and <color #ed1c24>impressum.html</color> to be adapted. The relative URLs should be specified. In relation to the page <color #ed1c24>links.html</color>page, the specifications for the href attribute would look like this:

<code html>
<!doctype html>
<html lang="de">
  <head>
    <title>Linksammlung</title>
    <meta charset="UTF-8">
  </head>
  <body>
    <nav>
      <a href="../index.html">Blog</a> |
      Links |
      <a href="about.html">Über mich</a> |
      <a href="impressum.html">Impressum</a>
    </nav> 
    <h1>Linksammlung</h1>
    <p>Interessante Linksammlung zu HTML-Themen</p>
  </body>
</html>
</code>

Navigate via the subfolder pages with <color #ed1c24>../</color> (here <color #ed1c24>../index.html</color>) to the parent folder, where <color #ed1c24>index.html</color> is located. The other two files <color #ed1c24>about.html</color>, <color #ed1c24>impressum.html</color> are located in the same folder as <color #ed1c24>links.html</color>so it is sufficient to specify only the file name. Similarly, you must also specify the files <color #ed1c24>about.html</color> and <color #ed1c24>impressum.html</color> files.

===== Insert links to other websites =====

Links to other websites are noted in the same way as links to pages on the same website. The difference lies in the absolute URL (i.e. the complete address) in the href attribute.

<code>
<nav>
  <ul>
    <li><a href="http://www.w3.org/html/wg/drafts/html/master/">HTML 5.1 Nightly</a></li>
    <li><a href="http://www.w3.org/">W3C</a></li>
    <li><a href="http://www.whatwg.org/">WHATWG</a></li>
  </ul>
</nav>
</code>

===== Open links in a new window ===== 

With the HTML attribute <color #ed1c24>target="_blank"</color> opens the link in a new window or tab.
The aim of using <color #ed1c24>target="_blank"</color> is not to lose the visitor to the page, but to leave the page open so that he returns there when he has read the page in the newly opened window or tab.

==== Example ====
<code html>
  <a target="_blank" href="http://www.w3.org/">Link in einem neuen Fenster öffnen</a>
</code>

If you were to activate the link text W3C in this example, the target address (here: http://www.w3.org) here) would actually be opened and loaded in a new window or tab. The aim of using the target="_blank" is of course primarily to open the

===== e-mail links =====

Email links start with <color #ed1c24>href="mailto:EMAIL-ADRESS"</color>. This opens the e-mail application with a specific e-mail address.

<code html>
  <a href="mailto:hans.muster@beispiel.ch">E-Mail senden</a> 
</code>

===== Phone links =====

Telephone links start with <color #ed1c24>href="tel:NUMBER"</color>. This is used to dial and call the number. (Desktop e.g. Facetime or online telephony)

<code html>
  <a href="tel:044 123 45 67">E-Mail senden</a> 
</code>

===== Download links =====

The download attribute enables links to be added as download links, regardless of the content type (MIME type).

<code html>
  <a href="dokument.pdf" download>Dokument herunterladen</a>
</code>

===== Anchor - set links to specific parts of a website =====

Nothing can be more annoying for a visitor than having to read a very long scientific treatise on a specific topic on a website and having to scroll up and down for a very long time to get to a specific section. For such cases, you can set so-called anchors with the global attribute id, which you can then jump to with an ordinary link with the a element. Such target anchors are used, for example, in Wikipedia for the content index of a topic.
To link to a specific area of a website, you only need:

  * an anchor (jump marker), which you can assign with the attribute <color #ed1c24>id="ankername"</color> attribute. For example: <color #ed1c24><h1 id="ankername">Heading xyz</h1></color>
  * a link that points to the anchor with <color #ed1c24>href="#ankername"</color> . To do this, the double cross character <color #ed1c24>#</color> is written in front of the anchor name. For example: <color #ed1c24><a href="#ankername">Jump to heading xyz</a></color>
